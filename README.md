Sistema de Gestión de Servicios del CAT

Este proyecto moderniza el proceso de soporte del Centro de Atención Tecnológica (CAT), centralizando las solicitudes y permitiendo un control claro del estado, prioridad y atención de cada caso. El objetivo es mejorar la organización, la trazabilidad y la calidad del servicio.

Objetivo

Implementar un sistema que permita registrar, asignar y dar seguimiento a solicitudes de soporte técnico de manera eficiente.

Funcionalidades

Registro de solicitudes de soporte.

Asignación de solicitudes a técnicos.

Gestión de estados: Pendiente, En proceso, Resuelto, Cerrado.

Control de prioridades.

Panel para técnicos y administradores.

Consulta del estado por parte del usuario.

Historial de acciones por solicitud.

Reportes y estadísticas básicas.
